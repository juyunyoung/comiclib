# 1. 개요

사용자가 자신이 읽은 만화책을 기록하고, 간단한 소감과 주요 등장인물들의 개별 순위를 매겨 저장·공유할 수 있는 모바일 앱을 개발한다.

# 2. 주요 목적

- 사용자만의 만화책 기록 일지
- 만화별 개성 있는 짧은 소감 남기기
- 등장인물별로 내가 정한 인기 순위 지정
- 기록 공유 및 회고, 소장 재미 제공


# 3. 주요 기능

## 1) 만화책 등록

- 만화 제목, 작가, 표지 이미지(직접 업로드 또는 검색), 완독/미완독, 읽은 날짜 입력
- 내 평점(5점 만점) 기능


## 2) 짧은 소감 작성

- 100자 내외의 간단한 서평/소감 작성란
- 좋았던 점, 아쉬웠던 점 등 체크박스(선택)


## 3) 등장인물 목록 및 순위 등록

- 등장인물 직접 입력 또는 추천 리스트에서 선택
- 각 인물별로 1~5순위 직접 지정(동일 순위 불가)
- 인물별 한 줄 코멘트 작성 가능


## 4) 검색 및 열람

- 내가 기록한 만화 목록, 소감, 등장인물 순위 열람
- 만화 제목/작가/인물명으로 검색
- 소감별 필터(평점순, 최근 등록순 등)
- 간단한 통계(읽은 만화 수, 최다 등록 작가, 내가 자주 1순위에 둔 인물 등)


## 5) 공유 기능

- 나의 만화 소감 및 인물순위 결과를 이미지/텍스트로 SNS, 메신저 등 공유 지원


# 4. 화면 예시

- 만화책 리스트 / 상세(소감 및 인물 랭킹 포함)
- 만화책 등록/수정
- 등장인물 순위(드래그\&드롭 등 UI)
- 내 통계


# 5. 기술 요구사항
- 간단한 백엔드(사용자별 데이터 저장)
- 사용자 인증(이메일, SNS)


# 6. 기타 고려사항

- 만화 DB 연동(검색용) + 사용자 직접 입력 허용
- 순위 시 بص각적 UI(예: 인물 캐릭터별 아이콘/프사)
- 개인정보 최소수집


# 7. 성공 지표

- 등록 만화 수, 사용자 월별 활동 수, 공유 횟수
- 소감/인물 순위 평균 등록 비율
